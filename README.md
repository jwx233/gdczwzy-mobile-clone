# 广东潮州卫职院官网移动端课程演示

基于目标学校官网公开页面制作的移动端前端复刻项目。项目仅用于课程演示，并在页面顶部明确标注“非官方网站”。

## 功能

- 移动端首页、新闻列表、公告列表、文章详情和本地搜索
- 全屏绿色抽屉菜单与二级栏目手风琴
- 新闻轮播、专题图片、友情链接、联系信息和回到顶部
- `npm run sync` 构建前同步公开内容，失败时保留最后一次快照
- GitHub Actions 每日同步并部署至 GitHub Pages

## 本地运行

```bash
npm install
npm run dev
```

同步公开内容：

```bash
npm run sync
```

测试和构建：

```bash
npm test
npm run build
```

## 说明

- 仅针对 `320px–768px` 移动端宽度设计。
- 原站外部微信文章直接跳转原链接。
- 内容和图片版权归原网站及对应作者所有。
- 本项目不实现智慧校园登录等敏感功能。
